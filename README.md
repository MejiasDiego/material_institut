# Sistema de Gestió de Material

Projecte Java per a la gestió d'inventari i préstecs de material a l'Institut Camí de Mar.

## Contingut
- Documentació d'usuari
- Documentació tècnica
- Projecte Java amb Javadoc

## Execució
1. Importar el projecte a Eclipse.
2. Executar la classe principal `InventariService`.
3. Consultar la carpeta `doc/` per al Javadoc.
